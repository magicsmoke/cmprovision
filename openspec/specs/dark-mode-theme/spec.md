### Requirement: Dark Theme Styling

All UI elements have appropriate dark variants.

#### Scenario: Dark mode active

- **WHEN** dark mode is active
- **THEN** page backgrounds use dark colors (e.g. gray-900)
- **AND** text uses light colors for readability (e.g. gray-100/200)
- **AND** cards, modals, and dropdowns use dark surface colors (e.g. gray-800)
- **AND** borders use dark-appropriate colors (e.g. gray-700)
- **AND** form inputs have dark backgrounds with light text
- **AND** buttons retain their semantic colors (blue, red, green) with dark-adjusted hover states
- **AND** alerts/notifications remain distinguishable with dark-appropriate backgrounds
